# BoardState Handoff Readiness

BoardState is prepared as the original advanced gameplay app and rules authority for the next ecosystem prompt series. Its rules engine, canonical shared-session contracts, save envelopes, sync namespaces, bridge adapters, and non-destructive migration tools are in place for BoardState Lite, Deck Nexus, and the future Hub to integrate against.

Current BoardState ownership remains: authoritative rules validation/resolution, Advanced Gameplay, mirrored Advanced views, Dry Run, tutorials, advanced saves, imported session snapshots, imported deck snapshots, waiver history, recovery saves, and legacy archive/export readiness.

Next app work:

- BoardState Lite must integrate with canonical shared sessions and the BoardState rules-engine adapter before live Simple/Advanced handoff is claimed.
- Deck Nexus must export immutable deck snapshots and validate deck legality through the shared contracts before live Deck Nexus linking is claimed.
- The future Hub must own ecosystem profiles, friends, tournaments, notifications, app linking, backups, and migration coordination before Hub linking or migration success is claimed.

Live cross-app integration is not claimed in this repository. Legacy data remains preserved, destructive migration remains disabled, and export/backup paths are readiness tools until destination apps accept the data.
