# BoardState UI Preservation Rules

The current visual design is locked.

Preserve:
- purple/dark fantasy dragon background
- transparent purple/black glass panels
- compact mobile companion layout
- old Life Tracker screen
- old Board State screen
- existing fonts, spacing, and buttons
- existing swipe/arrow navigation
- existing Game Options dialog style

Do not add:
- dashboard homepage
- desktop simulator UI
- always-visible opponent boards
- large menu home screen
- solid card theme replacing glass panels
- new full-page menu replacing the tracker/board

New systems should be hidden/service logic first. UI should only be exposed through compact overlays or existing dialogs.
